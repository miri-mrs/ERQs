# MIRI DATA DIRECTORY

The directory which holds the data for the JWST MIRI MRS analysis. 


## Getting Started

The data come from the [Barbara A. Mikulski Archive for Space Telescopes (MAST)](http://archive.stsci.edu/archive_news/2017/08-Aug/index.html) and the [STScI](http://archive.stsci.edu/jwst/simulations/index.html). 

Specifically, however, the [MIRI simulation](https://www.cosmos.esa.int/web/jwst/simulations) come from ESA.
